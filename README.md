# SUUMO 賃貸物件スクレイピング & データ前処理

東京23区の賃貸物件情報をSUUMOから収集し、分析用に整形するPythonプロジェクトです。

## 概要

一覧ページと詳細ページの2段階スクレイピングにより、物件の基本情報に加えて
入居可能時期・向き・設備・画像数などの詳細情報も取得します。
取得したデータは前処理スクリプトで住所分割・駅情報の正規化・設備フラグ化を行い、分析に利用しやすい形式に変換します。

## 機能

**スクレイピング（01_Scraping.py）**
- 東京23区の賃貸物件情報を取得
- 一覧ページ → 詳細ページの2段階アクセスで詳細情報を取得
- 区ごとにCSV出力 + 23区の平均家賃を自動集計
- サーバー負荷を考慮したレート制限（詳細ページ1.5秒 / 一覧ページ0.5秒）

**データ前処理（02-1_DataPreprocessing.py）**
- 住所の区・市・町丁目への分割
- 複数路線対応の駅情報解析（最短徒歩の駅を自動選択）
- 間取りのルーム数への数値変換
- 設備テキストから20項目のフラグ生成
- 型変換ユーティリティ（NaN・None・型不一致を一括処理）

## セットアップ


## 使い方

```bash
# Step 1: スクレイピング実行（01_OutputData/ にCSVが生成されます）
python 01_Scraping.py

# Step 2: データ前処理実行（02_PreprocessingData/ に整形済みCSVが生成されます）
python 02-1_DataPreprocessing.py
```

## 取得データ項目

| カテゴリ | 項目 |
|----------|------|
| 基本情報 | 物件名・家賃・管理費・間取り・面積・築年数 |
| 所在地 | 住所（区・市・町丁目）・階数 |
| 駅情報 | 路線名・駅名・徒歩分数 |
| 詳細情報 | 向き・角部屋・最上階・入居可能時期 |
| 設備 | オートロック・宅配BOX・エレベーター など20項目 |
